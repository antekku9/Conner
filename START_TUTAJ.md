## 💡 Szybkie odpowiedzi

**Q: Jak najszybciej uruchomić stronę?**  
A: `EKSPORT_Z_FIGMA_MAKE.md` → Publish → GitHub → 2 minuty gotowe!

**Q: Czy muszę znać programowanie?**  
A: Nie! Do eksportu z Figma Make nie potrzebujesz kodu. Do edycji produktów - podstawowa znajomość plików.

**Q: Czy GitHub Pages jest darmowy?**  
A: Tak! Całkowicie darmowy dla projektów publicznych.

**Q: Czy mogę użyć własnej domeny (conner.pl)?**  
A: Tak! Instrukcja w `GITHUB_DEPLOYMENT.md`

**Q: Jak dodać nowy produkt?**  
A: Edytuj plik `src/app/data/products.ts` - instrukcja w `INSTRUKCJA_EDYCJI.md`

**Q: Jak zmienić banery?**  
A: Edytuj plik `src/app/components/conner/BannerSlider.tsx` - instrukcja w `INSTRUKCJA_EDYCJI.md`

**Q: Czy strona ma system RMA (wizyty serwisowe)?**  
A: Nie w tej wersji. System RMA wymaga bazy danych (Supabase). To można dodać w przyszłości.
