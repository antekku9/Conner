  JAK DZIAŁA 404.html REDIRECT
═══════════════════════════════════════════════════════════════════

  Przykład:
  ┌────────────────────────────────────────────────────────────────┐
  │ 1. Wchodzisz: antekku9.github.io/Conner/produkty             │
  │ 2. Naciskasz F5 (refresh)                                     │
  │     ↓                                                          │
  │ 3. GitHub Pages szuka: Conner/produkty/index.html             │
  │ 4. Nie znajduje → serwuje 404.html                            │
  │     ↓                                                          │
  │ 5. 404.html uruchamia skrypt JavaScript                       │
  │ 6. Przekierowuje do: antekku9.github.io/Conner/?/produkty    │
  │     ↓                                                          │
  │ 7. index.html się ładuje (React app)                          │
  │ 8. React Router parsuje: /?/produkty → /produkty              │
  │     ↓                                                          │
  │ 9. ✅ Pokazuje ProductsPage!                                  │
  └────────────────────────────────────────────────────────────────┘

  Magiczne! ✨
